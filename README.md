# Invitación Juana 15 Años

Proyecto de invitación digital web responsiva, optimizada para compartir vía WhatsApp.

## Tecnologías Utilizadas
- HTML5 Semántico
- CSS3 (Variables, Grid/Flexbox, Animaciones Nativas)
- JavaScript Vanilla (ES2025)

## Estructura del Proyecto
- `index.html`: Estructura principal.
- `css/`: Estilos modulares (variables, layout, animaciones, responsive).
- `js/`: Lógica para el contador, formulario RSVP y animaciones al scrollear.
- `assets/`: Imágenes optimizadas y archivos tipográficos (en progreso).

## Desarrollo Local
1. Clona el repositorio: `git clone https://github.com/mis15juana/mis15juana.github.io.git`
2. Abre la carpeta en VS Code u otro editor.
3. Utiliza la extensión "Live Server" o un servidor HTTP local para previsualizar los cambios.

## Deploy a GitHub Pages
1. Asegurate de hacer commit y push a la rama principal (`main` o `master`).
2. En GitHub, ve a **Settings > Pages**.
3. Selecciona la rama principal como fuente.
4. El sitio estará disponible en `https://mis15juana.github.io/`.
